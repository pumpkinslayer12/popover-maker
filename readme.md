## Description

Popover Maker allows you to use the block editor to add a simple popover to any wordpress page or post, leveraging existing blocks.

## Installation

1. Upload the plugin files to the `/wp-content/plugins/popover-maker` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
